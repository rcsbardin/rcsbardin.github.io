Lavoyger using Able Player for Vimeo
With Captions via Local VTT Files

This page is a test of using locally hosted caption files - WebVTT with a Vimeo hosted video file using the Able Player. Why do this? Because Vimeo's player doesn't support caption placement and styling which are important for this film. Default caption placement will cover on screen titles and we need a format that supports overlapping / simultaneous dialog and sound descriptions
